Все секции сделаны через тег <section>.

Для того, чтобы при прокрутке страницы она автоматически докручивалась до нужного блока, необходимо повесить на блок атрибут data-anchor="slide".
Чтобы при докрутке блок выравнивался по центру страницы, необходимо повесить атрибут data-anchor="case".

Всплывающее окно с информацией о проекте реализовано через один блок, в котором динамически подменяется контент одного проекта на контент другого.
Информация, которая вставляется во всплывающее окно, находится внутри самой карточки проекта (class='project-item') в блоках:
— h3.project-intro (Заголовок)
— div.popup-info (Скрытый блок с информацией)
—— p.popup-info--text (Текст во всплывающем окне)
—— p.popup-info--whatwedid (Текст для блока What We Did)
—— div.popup-info--images (Изображения для всплывающего окна)

Основная картинка проекта (карточки проекта) находится в блоке div.project-image.
